<br>
只要使用一个32位数组bit(32位已经足够表示题目给出的最大数字)模拟数字在计算机中存储的位，每个数字转换为2进制之后，对应的位在数组bit中相应的位置进行+1操作。<br>
遍历完所有数之后，返回bit中最大的值ans，因为这个值ans表示，有ans个数字在这个位上都是1。
